# NCHU AllPass

NCHU AllPass 是一個專為國立中興大學學生設計的智慧校園助理 App，目標是把分散在校務系統、iLearning、課表、行事曆與校園生活服務中的資訊整合到單一入口，讓學生能用更直覺的方式掌握課業進度、安排行程，並做出更好的修課與生活決策。

本專案為「智慧物聯網」課程第 14 組期中專案提案。

## 專案動機

中興學生在日常校園生活中，經常需要在多個獨立系統之間切換，例如校務系統、iLearning、課程查詢、行事曆與其他校園服務。這些系統功能分散、登入流程重複，且多數官方入口仍以傳統網頁為主，行動端操作體驗不佳。

NCHU AllPass 希望解決的核心問題不是單純「做一個選課工具」，而是建立一個真正以學生情境為中心的整合型行動助理，降低資訊碎片化帶來的時間成本與認知負荷。

## 核心目標

- **一站式資訊整合**：將課表、學期事件、自訂行程與校園服務入口集中管理。
- **行動端優先體驗**：以跨平台 App 提供 iOS 與 Android 一致、流暢的操作體驗。
- **主動式學習輔助**：透過通知、提醒與狀態整理，協助學生掌握作業、考試與重要時程。
- **動態學業模擬**：提供 GPA 試算與 What-if Analysis，讓學生評估不同成績或修課組合對整體學業表現的影響。
- **降低系統切換成本**：減少反覆登入與跨平台查找資訊的摩擦。

## 主要功能規劃

### 行事曆模組

整合學期事件、課程時程與使用者自訂行程，讓學生能以單一行事曆掌握每週與整學期的安排。未來可搭配推播提醒，避免錯過選課期限、考試時間或作業截止日。

### 一週課表

以行動端友善的版面呈現每週固定課程與自訂行程，讓學生能快速確認當天課程、教室與時間安排。

### 個人中心

提供個人化設定、帳號狀態與常用服務管理，作為使用者在 App 中管理校園資訊的集中入口。

### 工具箱

預計整合多個進階工具：

- 成績管理與 GPA 動態試算
- 優化的課程搜尋介面
- 數位點名整合
- 圖書館與常用校園服務入口
- 自訂行事曆推播
- 美食轉盤等校園生活小工具

## 與現有方案的差異

相較於官方校務系統，NCHU AllPass 更重視資料整合與行動端體驗。官方系統通常提供靜態資料查詢，且資訊散落在不同入口；本專案則希望把資料結構化後集中呈現，並加入學業規劃與生活輔助功能。

相較於一般行事曆工具，NCHU AllPass 不只是時間管理工具，而是針對中興學生需求設計的校園原生平台，能理解課表、學期事件、學業成績與校務流程之間的關聯。

相較於既有的 NCHU Unofficial App 或 AI 興學伴，本專案的定位更偏向個人化、主動式、跨服務整合的 Native App，目標是從單向資訊讀取延伸到可互動、可規劃、可提醒的 All-in-One 校園助理。

## 技術方向

- **前端框架**：Flutter
- **目標平台**：iOS、Android
- **設計重點**：跨平台一致性、行動端 UI/UX、資訊層級清晰、操作流程簡化
- **資料處理方向**：以本地端處理與安全儲存為優先考量，降低不必要的第三方資料暴露

由於學校目前缺乏完整公開 API，且登入流程可能包含人機驗證等限制，資料擷取與狀態同步會是本專案的重要技術挑戰之一。

## 預期成效

NCHU AllPass 預期能降低學生在不同校園系統間切換與查找資料的成本，讓課表、成績、行程與生活資訊能在同一個 App 中被理解與管理。透過 GPA 試算、結構化課表與主動提醒，學生可以從被動接收資訊，轉向主動規劃自己的學習與校園生活。
